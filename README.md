# 🧮 Scientific Calculator

A modern and responsive **Scientific Calculator** built using **HTML, CSS, and JavaScript**.  
This project supports both basic and advanced mathematical operations with a clean UI and smooth user experience.

---

## 🚀 Features

- ✅ Basic Operations (+, -, ×, ÷, %)
- ✅ Scientific Functions:
  - sin, cos, tan (Degree based)
  - log (base 10)
  - square (x²)
  - power (xʸ)
  - square root (√)
  - π (Pi value)
- ✅ Responsive Design (Mobile + Desktop)
- ✅ Dark / Light Mode Toggle 🌙☀️
- ✅ Error Handling (invalid input, negative roots, etc.)
- ✅ Clean and modern UI

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
